1．課題
	8bitSWの0がONのとき，0.5秒周期で２つのLEDを同時点滅，
	8bitSWの0がOFFで，8bitSWの1がONのとき，1秒周期で２つのLEDを同時点滅，
	8bitSWの0と1がOFFで，8bitSWの2がONのとき，２秒周期で２つのLEDを同時点滅
	それ以外の組み合わせでは常時LEDは消灯
	実行の様子を観察し，その様子を実行結果として報告しなさい
	
2．使用したプログラム
	/**********************************************************
	マザーボードの8ビットスイッチがつながっているP2の各ビットを
	直接見てみよう。
	また，関数get8BitSW()が得る値を見てみよう。
	**********************************************************/
	#include <3048fone.h>
	#include "h8_3048fone.h"

	main()
	{
		unsigned char sw1,sw2;
		initLed();
		initSCI1();
		init8BitSW();/*8bitSWの初期化*/
		while(1) {
			sw1=P2.DR.BYTE;
			sw2=get8BitSW();
			SCI1_printf("P2.DR.BYTE=%08b get8BitSW()=%08b\n",sw1,sw2);
		}
	}

4．考察
	プログラムから見える1／0は，ポートを電圧を測定し直接見たときに見える1／0と逆であることが分かった．
	プログラム上でプルアップ設定を行っているため，上の表の結果になる．
5．まとめ
	実際に電圧を測定することにより，マイコン内部から見えるON／OFFは，ポートに流れる電圧のON/OFFのレベル（電圧）と逆である．
	つまり，スイッチが1とプログラムで取得されているとき，そのスイッチを指すポートには0を表すレベルの電圧がかかっている．
