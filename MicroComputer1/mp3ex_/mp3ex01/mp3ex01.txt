1．課題
	周期1秒，デューティ比50%で両方のLEDが同時に点滅するプログラム（mp3ex01.txt）
	ただし，周期とは点灯開始時刻から次の点灯開始時刻までの時間のことであり，
	デューティ比とは1周期中の点灯時間の割合のことである。例えば周期8秒の点滅で，
	点灯時間が2秒だった場合はデューティ比25%である。
	実行の様子を観察し，その様子を実行結果として報告しなさい。

2．使用したプログラム
	#include <3048fone.h>
	#include "h8_3048fone.h"

	void waitmsec(int msec)
	{
		int i,j;
		for (i=0;i<msec;i++) {
			for (j=0;j<4190;j++);    /*4190は実測によって求めた値 25MHz駆動*/
		}
	}

	main()
	{
		initLed();
		while(1) {
			turnOnLed(0); /*LED0のON*/
			turnOnLed(1); /*LED1のOFF*/
			waitmsec(500);
			turnOffLed(0); /*LED0のOFF*/
			turnOffLed(1); /*LED1のON*/
			waitmsec(500);
		}
	}
3．実行結果
	LED0とLED1が両方点灯
	0.5秒経過
	LED0とLED1は両方消灯
	0.5秒経過
	LED0とLED1が両方点灯
	0.5秒経過
	LED0とLED1は両方消灯
	以下，上の動作を繰り返す．
	
	周期：1秒
	デューティ比50%

4．考察
	点滅周期は今回の場合whileループの中である．
	今回は周期1秒のため，
	whileループ内のwaitsec()の合計を1000にすればよい．
	そこから逆算し，プログラムを作成した．

5．まとめ
	turnOnLED()及びturnOffLED()の使い方を覚えた．
	また，「関数の中には，P5のDRの第nビットを0(or1)にする」というコマンドの
	簡略化であり，使いやすくしたものだということを理解した．
